# Type Script
- 
## Iniciando o projeto

```
npm init -y
```

- TypeScript
```
npm i typescript -D
```

- ts-node
```
npm i ts-node -D
```

- ESlint
```
npm i eslint -D
```

- Faz a integração do ESlint com o typescript:
```
npm i @typescript-eslint/eslint-plugin @typescript-eslint/parser -D
```

- Prettier
```
npm i prettier eslint-config-prettier eslint-plugin-prettier -D
```

```
npx tsc --init
```
